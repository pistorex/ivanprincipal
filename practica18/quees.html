QUE ES

CSS son las siglas de "Cascade StyleSheet". Se trata de una especificación sobre los estilos físicos aplicables a un documento HTML, y trata de dar la separación definitiva de la lógica (estructura) y el físico (presentación) del documento.

Objetivos
  1. Dar formato páginas web
  2. Separar diseño – contenido
  3. Centralizar propiedades aspecto
  4. Facilitar mantenimiento diseño

Características
  1. Fichero de texto
  2. Formato de páginas HTML, PHP, ...
  3. Herencia de propiedades entre elementos (jerarquía)
  4. Clases definibles por el usuario
